Accessible UI, Better User Experience

Use headings such as h1, h2, etc. in the UI to make it accessible. Attached Figma sshot - "Text Styles" panel marked in red border shows Heading 1 style selected.

In the above Figma sshot - "Read More" button text - 19px bold, color (#FFFFFF) passed with background color (#F05537) with 3.5:1 contrast ratio for Large text and "Read More" link text in the paragraph - 16px regular, color (#F05537) failed with background color (#FFFFFF) with 3.5:1 contrast ratio for Normal text. Minimum requirement ratio to pass CCA test is 4.5:1 for normal text.

In the above Figma sshot - All the informative icon images marked with red border must meet 3:1 contrast ratio to pass CCA test. "Arrow" icon (#999999) - pointing downward used in the accordion here failed CCA test with background color (#fffff).

In the above Figma sshot - Text input field border (#999999) failed CCA test with background color (#fffff). Also image of text "Donate Now.." (#fffff) failed CCA test with background color (#F7AA9B). Must meet 3:1 contrast ratio to pass CCA test.
