Headings

Use heading level such as <h1> to <h6> while designing UI in Figma to make the page structure accessible.

In the attached Figma sshot - H1 under the "Text Styles" panel marked in red border shows the Heading 1 style selected.

Also in sshot, "Accessible UI, Better User Experience" text marked as <h1>. "UI elements" text marked as <h2> and "Headings", "Color", "Images", "Non-text Contrast" text marked as <h3> as per the structure of page.

Color Contrast

When color is used on a web page, to ensure accessibility, the right color contrast must always be identified.

WCAG 2.0 level AA requires a contrast ratio of at least 4.5:1 for normal text and 3:1 for large text.

In the attached Figma sshot - the "Read More" button text - 19px bold, passed the CCA test for Large text. "Read More" link test in the paragraph - 16px regular, failed CCA text for Normal text.
